# Easy-cli
A easy way to create linter/release/git standard cli

## Usage
```
npx @preflower/cli
```

## Standard

### Javascript Standard
easy-cli js base on `standard`

### CSS Standard
easy-cli css base on `stylelint-config-sass-guidelines`

### Git Standard
easy-cli git base on `angular`
